# Карта сайта

```
`/` - Главная станица
├── `purpose` - Цель
├── `whitepaper` - Whitepaper
├── `roadmap` - Дорожная карта
├── `faq` - Вопросы-Ответы
├── `contacts` - Контакты
└── `legal` - Юридическая информация
    ├── `terms` - Условия ипользования
    └── `privacy` - Политика конфиденциальности
        └── `cookies` - Использование файлов cookie
```
